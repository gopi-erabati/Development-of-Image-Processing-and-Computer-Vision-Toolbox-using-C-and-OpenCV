To run this tool box

1. You need to have openCV 3.2 
	and also contribution files to run feature matching(SURF, SIFT) and to find Fundamental matrix and homography

2. This code is developed in Qt IDE.
	Qt 5.9.0
	Compiler : MinGW 5.3 32bit
	with cmake

3. Open Qt and open CVToolBox.pro file.

4. In 'CVToolBox.pro' file change the path to include opencv files according to your system location of opencv.

5. Change the build directory to build the code

6. Build and Run the code to start the tool box!

Cheers!
